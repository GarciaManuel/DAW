Database vs DMS

  • ¿Qué es entonces una base de datos - DB?
    R= Colección de datos persistentes que son usados por una empresa mediante la aplicación de sistemas.
  • ¿En qué casos es conveniente usar bases de datos?
    R= Siempre y cuando se necesiten mantener datos de manera persistente, que se busquen almacenar achivos de manera compacta. Por otra parte es sumamente necesario al necesitar acceso de manera rapida y eficaz a grandes cantidades de información ademas cabe resaltar que para tener archivos actualizados al dia y disponibles en tiempo real.
  • ¿Qué es un sistema de gestión de base de datos - DBMS?
    R= Es una coleccion de programas que permitena los usuarios el crear y mantener una base de datos. El proposito general de un DBMS es el facilitar el proceso de definir, construir, manipular y compartir bases de datos entre varios usuaarios y aplicaciones.
  • Enlista y explica las funciones/responsabilidades que tienen los DBMS.
    1. Definicion: Especificacíon de tipos de datos, estructuras y colecciones de datos a guardar en la base de datos, la definicion o descripcion de informacion tambien es guardada como un catalogo o diccionario, llamado "metadatos".
    2. construcción: Proceso de almacenamiento de informacion en algun medio controlado por la DBMS.
    3. Manipulación: Una base de datos incluye funciones como: "consultar" con el fin de recabar información, "actualizar" y "generar" reportes.
    4. Compartir: Permite a la base de datos el acesso a multiples usuarios simultaneamente
    5. Protección: Incluye proteccion contra fallas de hardware o software ademas de seguridad para accesos no autorizados
    6. Mantenimiento: Debe permitir a la base de datos el realizar cambios para satisfacer las necesidades de acuerdo al tiempo
  • Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.
    R= En el proyecto de videojuegos de POO ya que requeriamos el lograr almacenar informacíon de manera que el usuario pudiera mantener su score, datos y logros pese a cerrar la aplicación.
    Por otra parte pudo haber sido de gran apoyo en Organización Computacional ya que nos hubiese permitido crear un aparato con mas funcionalidades que permitiera a la maquina recordar comando, instrucciones y/o peticiones.
