  • ¿Cómo es que si el código está en un archivo con código html que se despliega del lado del cliente, se ejecuta del lado del servidor? Explica.
    R= Ya que el usaurio al buscar/conectarse a una pagina web lo que esta realizando es solicitarle al servidor por medio de un HTTP Request, la informacion contenida en ese pagina que a la vez se ocnecta con un archivo, este contendra la informacion que se desea buscar en un archivo de formato HTML ya que es necesario que el navegador lo pueda comprender.
