Type Methods Error Codes
HTTP
  • GET: Retrieve data
  • HEAD: Get request but without response in the body
  • POST: Submit data
  • PUT: Update data
  • DELETE: Delete resource
  • CONNECT: Create a tunnel to the server id by target source
  • OPTIONS: Describe the communications options for the resource
  • TRACE: Message loop-back test along the path to the target resource
  • PATCH: Apply partial modifications to the resource
 Common errors:
  • 200-OK
  • 201-Created
  • 202 Accepted
  • 401-Unauthorized
  • 404-Not Found
  • 408-Request Timeout
